아이디어 구체화

체택한 아이디어!!
: 국가 공공기관에서 하는 프로그램들을 사용자 위치를 기반으로 정리하여 알려주는 사이트

Step1. 문제 인식

1. 국가에서 운영하기에 가격이 비교적 저렴함에도 불구하고, 사람들의 참여도가 낮다.
2. 구민센터, 동사무소 등에서 하는 프로그램들에 대한 홍보부족
: 주로 건물 게시판에 붙어있거나, 구청사이트에 올라와있다. 다만, 구청 사이트는 구성이 복잡하여 찾기가 힘들다.

Step2. 웹사이트 구성

▷간단한 navigation : 장소 누르면 -> 해당 장소에서 진행하는 프로그램 목록 -> 프로그램 이름, 기간, 가격, 인원, 신청기간 등 정보 보여주기
▷detail : 장소는 7~10개. 각 장소마다 하는 프로그램 5~8개
